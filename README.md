## What Is SHA-256?

SHA-256 is a cryptographic hash function that:

- Produces a fixed-length hash
- Detects file modifications
- Is widely used in cybersecurity
- Is part of the SHA-2 family

---

## Practical Uses

- Verify downloads
- Detect file tampering
- Backup verification
- Security audits
- Malware analysis
